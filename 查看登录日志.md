# 查看登录调试日志

## 📝 操作步骤

1. **尝试登录**
   - 在浏览器访问: http://localhost:5001
   - 输入用户名: `admin`
   - 输入密码: `admin123`
   - 点击登录

2. **查看调试日志**
   ```bash
   docker logs --tail 50 project-management-app
   ```
