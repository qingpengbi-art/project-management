# 🌍 内网穿透访问方案

## 🛠️ 使用ngrok（推荐）

### 安装ngrok
```bash
# 方法1：使用Homebrew
brew install ngrok/ngrok/ngrok

# 方法2：直接下载
# 访问 https://ngrok.com/download 下载Mac版本
```

### 启动内网穿透
```bash
# 穿透前端服务
ngrok http 3001

# 穿透后端服务（新终端）
ngrok http 5001
```

### 获取外网地址
ngrok会提供类似这样的地址：
- 前端：https://abc123.ngrok.io
- 后端：https://def456.ngrok.io
